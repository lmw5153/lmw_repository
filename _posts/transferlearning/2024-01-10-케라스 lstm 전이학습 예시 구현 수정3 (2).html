def MSE(y_pred, y_true):
    return np.mean((y_true - y_pred)**2)


def MAE(y_pred, y_true):
    absolute_errors = np.abs(y_true - y_pred)
    mae = np.mean(absolute_errors)
    return mae

def MAPE(y_true, y_pred):
    epsilon = 1e-10
    percentage_errors = np.abs((y_true - y_pred) / (y_true + epsilon)) * 100
    percentage_errors = np.nan_to_num(percentage_errors)
    mape = np.mean(percentage_errors)  
    return mape

학습 시행 1번 반복에 따른 결과입니다¶

  • 전이학습으로 학습한 모델은 매우 빠른 학습시간을 보여준다
  • MSE, MAE, MAPE 지표는 target data를 fitting한 모델이 우수하다. 그러나 학습시간대비 전이학습모델 또한 성능이 매우 훌륭함
  • freezing model 보다 non freezing model이 학습속도가 근소하게 증가하는편이다.
  • lstm보다 gru가 전이학습 모델이 더 우수한 것을 알 수 있다. 또한 학습속도 또한 근소하게 빠름
  • gru는 target data를 fitting한 모델과 non freezing 전이학습 모델 간 성능이 큰 차이가 없다.

-추가

  • 전이학습 모델은 학습 속도가 매우빨라 patience 옵션을 조정해도 될것 같습니다.patience 옵션이 증가할수록 non_freezing model이 use_freezing model보다 성능이 좋은 것을 확인했습니다

  • 모든 레이어를 freezing한 model은 학습 시행마다 결과 성능의 폭이 큰 차이로 벌어지는 것을 확인했습니다. 현재는 유의한 성능 차이가 보이지 않지만 다시 학습을 시행했을 경우에 유의한 성능차이가 나타날 수 있습니다.
